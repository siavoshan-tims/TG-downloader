# TG-downloader


## نحوه استفاده نهایی:

1. **ریپازیتوری جدید** در گیت‌هاب بسازید
2. فایل‌های بالا رو با همان ساختار پوشه‌ها اضافه کنید
3. برید به تب **Actions** و workflow رو اجرا کنید

این ابزار کاملاً رایگان و بدون هیچ لایسنس یا محدودیتی کار می‌کنه! فقط کافیه لینک پست عمومی تلگرام رو بدید تا خودکار دانلود بشه.
